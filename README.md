## 🚀 Features

- 🔍 Search cybersecurity-related news  
- 🎛 Filter articles based on recency  
- 🔃 Sort articles by date or popularity  
- ⭐ Save favorite articles (localStorage)  
- 🌙 Clean and responsive UI  

---

## 🧱 Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- Fetch API  

---

## 🌐 API Used

- Hacker News API  

https://hn.algolia.com/api/v1/search?query=cybersecurity

---

## 📌 Overview

Henix is a simple web-based project that helps users stay updated with cybersecurity news.  
It demonstrates API integration, dynamic UI rendering, and use of JavaScript array methods like `map`, `filter`, and `sort`.

---

## ⚙️ How to Run

1. Clone the repository  
2. Open `index.html` in your browser  
3. Start exploring
